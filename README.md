# Утилита анализа файловой системы

## Запуск

Из корневой папки проекта:

```
python src/main.py <путь_к_папке>
```

Пример:

```
python src/main.py .
```

## Проверка резервной копии

```
python src/main.py source backup
```

где:

* source — исходная папка;
* backup — папка резервной копии.

## Возможности

* обход файловой системы;
* сохранение информации о файлах в SQLite;
* вычисление SHA-256 хэшей;
* поиск дубликатов;
* сравнение резервной копии;
* определение отсутствующих файлов;
* определение лишних файлов;
* определение изменённых файлов;
* сохранение истории проверок в SQLite.
