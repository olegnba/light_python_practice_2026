# Утилита анализа файловой системы

## Запуск из корневой папки проекта

Анализ всех файлов в папке:

```bash
python src/main.py <путь_к_папке>
```

Пример:

```bash
python src/main.py .
```

## Фильтрация по расширению

Анализ только файлов указанного типа:

```bash
python src/main.py <путь_к_папке> <расширение>
```

Примеры:

```bash
python src/main.py source .txt
```

```bash
python src/main.py source .png
```

## Проверка резервной копии

```bash
python src/main.py source backup
```

где:

* `source` — исходная папка;
* `backup` — папка резервной копии.

## Проверка резервной копии с фильтрацией

```bash
python src/main.py source backup .txt
```

Пример:

```bash
python src/main.py source backup .png
```

В этом режиме анализируются только файлы указанного расширения.

## Возможности

* рекурсивный обход файловой системы;
* фильтрация файлов по расширению;
* сохранение информации о файлах в базе данных SQLite;
* вычисление SHA-256 хэшей файлов;
* поиск дубликатов файлов;
* сравнение исходной папки с резервной копией;
* определение отсутствующих, лишних и изменённых файлов;
* сохранение истории проверок резервной копии
